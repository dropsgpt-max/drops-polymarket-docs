# ❌ Ошибки

## **Ошибки в Drops Bot: Причины и Решения**

При использовании функций Drops Bot, связанных с блокчейном — таких как свапы, отслеживание ликвидности и взаимодействие со смарт-контрактами — могут возникать ошибки. Ниже собраны самые частые из них, их причины и способы устранения.

### **1. Управление проскальзыванием (Slippage)**

{% hint style="danger" %}
#### Ошибка

#### **«Failed due to ‘executing swap was failed on the blockchain. Reason: Exceeds desired slippage limit’»**
{% endhint %}

{% tabs %}
{% tab title="📍 Причина" %}
Указано слишком низкое значение проскальзывания. Рыночная цена изменилась сильнее, чем разрешено настройками.
{% endtab %}

{% tab title="💡 Что такое проскальзывание?" %}
Разница между ожидаемой и фактической ценой исполнения транзакции. Помогает защититься от волатильности рынка.
{% endtab %}

{% tab title="✅ Решение" %}
Увеличьте допустимое значение проскальзывания в [настройках](settings.md), чтобы снизить риск неудачного исполнения транзакции.
{% endtab %}
{% endtabs %}

### **2. Необходимость в резерве для Solana**

{% hint style="danger" %}
#### Ошибка

#### **«Failed due to 'Balance is not enough to transfer and cover Solana fees’»**
{% endhint %}

{% tabs %}
{% tab title="📍 Причина" %}
Недостаточно SOL для покрытия комиссии.
{% endtab %}

{% tab title="✅ Решение" %}
Для успешного выполнения транзакций необходимо иметь на счету дополнительные **0.01 SOL** в качестве резерва. Эти средства предназначены для покрытия возможных неожиданностей в процессе исполнения транзакций.
{% endtab %}
{% endtabs %}

### **3. Аккаунт заморожен (Frozen Account)**

{% hint style="danger" %}
#### Ошибка

#### **«Executing swap was failed on the blockchain. Reason: account is frozen»**
{% endhint %}

{% tabs %}
{% tab title="📍 Причина" %}
На кошельке есть подозрительный или вредоносный токен. Это может быть:

* Токен с уязвимостью или вредоносным кодом
* Скам-актив
* Неавторизованный токен, внесённый в чёрный список
{% endtab %}

{% tab title="✅ Решение" %}
* Проверьте список токенов
* Удалите всё подозрительное
* Не взаимодействуйте с непроверенными активами
* **Создайте новый кошелёк** и переведи на него только безопасные активы
{% endtab %}

{% tab title="🔒 Рекомендации" %}
Проверяйте токены через **Token Sniffer**, **DexTools** или **CoinGecko** перед использованием.
{% endtab %}
{% endtabs %}

### **4. Ошибочный адрес получателя**

{% hint style="danger" %}
#### Ошибка

#### **«Failed due to ‘there was an error while executing transfer’»**
{% endhint %}

{% tabs %}
{% tab title="📍 Причина" %}
* Неверный или несуществующий адрес
* Ошибки в структуре
* Несовпадение сети
{% endtab %}

{% tab title="✅ Решение" %}
* Убедитесь, что адрес корректен
* Проверьте активность кошелька через **обозреватель блокчейна**
* Не вставляйте адреса “вслепую” — только из доверенных источников
{% endtab %}
{% endtabs %}

### **5. Не удалось построить маршрут обмена**

{% hint style="danger" %}
#### Ошибка

#### **«Failed due to ‘failed to build swap route details’»**
{% endhint %}

{% tabs %}
{% tab title="📍 Причина" %}
Jupiter не может найти маршрут между токенами. Возможные причины:

* Недостаток ликвидности
* Отсутствие пула
* Токены не поддерживаются
* Слишком большая/маленькая сумма
{% endtab %}

{% tab title="✅ Решение" %}
* Проверьте токены в **Jupiter** вручную
* Уменьшите сумму обмена
{% endtab %}
{% endtabs %}

### **6. Транзакция не подтверждена**

{% hint style="danger" %}
#### Ошибка

#### **«Failed due to ‘tx (tx hash) was not confirmed, please try to increase fee or contact support’»**
{% endhint %}

{% tabs %}
{% tab title="📍 Причина" %}
* Низкая приоритетная комиссия
* Перегруженная сеть
{% endtab %}

{% tab title="✅ Решение" %}
* Увеличьте значение **Priority Fee** в настройках DropsBot
* Повторите транзакцию
* Следите за текущей загруженностью сети через сторонние сервисы (например, **Etherscan Gas Tracker**)
{% endtab %}
{% endtabs %}

### **7. Ошибка при обработке транзакции**

{% hint style="danger" %}
#### Ошибка

#### **«There was an error while processing transaction»**
{% endhint %}

{% tabs %}
{% tab title="📍 Причина" %}
Универсальная ошибка, возникающая по разным причинам:

* Сбои в сети
* Ошибки в смарт-контракте
* Недостаток средств
* Неверные параметры транзакции
* Ошибки DEX или маршрутизатора
{% endtab %}

{% tab title="✅ Решение" %}
* Повторите транзакцию
* Проверьте баланс и все параметры
* Убедитесь в стабильности сети
* Если ошибка сохраняется — смените токены, сеть или обратитесь в поддержку
{% endtab %}
{% endtabs %}

### ✅ Общие рекомендации:

* Не работайте с непроверенными токенами
* Проверяйте ликвидность и маршруты заранее
* Следите за загруженностью сети
* Пользуйтесь только надёжными кошельками и адресами
* В случае подозрительных ошибок — пересоздайте кошелёк и переведите активы вручную
