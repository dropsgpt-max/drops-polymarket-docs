# 👛 Your Wallets

#### 🔹 **Основной блок кошельков**

Каждый подключенный кошелёк отображает следующую информацию:

* **Your Wallets List** – Список всех добавленных кошельков для торговли
* **Название кошелька (кликабельно)** – Открывает страницу кошелька на **Solscan**
* **/EDIT** – Ссылка для перехода в меню редактирования конкретного кошелька
* **Адрес кошелька** – Публичный адрес в сети Solana, используемый для хранения и трейдинга
* **Balance:** – Текущий баланс в SOL и его эквивалент в USD
* **Positions:** – Общее количество токенов, находящихся на кошельке

***

#### 🔘 **Кнопки действий с кошельками**

* **➕ Generate Wallet** – Создание нового кошелька прямо внутри бота
* **↙️ Import Wallet** – Импорт одного или нескольких кошельков с помощью приватных ключей
* **✏️ Edit All Wallets** – Массовое редактирование всех добавленных кошельков
* **↩ Back** – Возврат в основное меню трейдинга

***

#### ✏️ **Редактирование кошелька (/EDIT)**

После нажатия на **/EDIT**, становятся доступны следующие опции:

* **✏️ Rename Wallet** – Переименование кошелька
* **📩 Export Private Key** – Экспорт приватного ключа (будьте осторожны!)
* **🗑 Delete** – Удаление кошелька из списка торговых
* **↩ Back** – Возврат в главное меню трейдинга

***

### 🔄 **Как импортировать кошелёк**

{% stepper %}
{% step %}
Откройте меню трейдинга командой **/trading**
{% endstep %}

{% step %}
Перейдите в раздел **👛 Your Wallets**
{% endstep %}

{% step %}
Нажмите **↙️ Import Wallet**
{% endstep %}

{% step %}
Отправьте приватный ключ

* Также возможно импортировать **несколько ключей**, указав каждый с **новой строки**
* Либо загрузите файл формата **.txt** или **.csv**
  * Максимальный размер файла: **100 KB**
{% endstep %}
{% endstepper %}

{% hint style="success" %}
Импортированные кошельки появятся в интерфейсе трейдинга и будут готовы к использованию.
{% endhint %}
