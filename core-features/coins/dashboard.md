# ✨ Dashboard

### **Accessing the Coin Dashboard**

* Select **“🪙 Coins”** from the inline keyboard, or
* Send the command **/coins** in chat.

***

### Token Names and Prices

* This menu displays all the tokens currently being tracked. The **current price** of each token is shown.

### Price Change in the Last Hour

* Next to each token's price, you can see its **percentage change** over the selected time interval.

### Price Change Charts

* Each token includes a **link to a price change chart** for detailed trend analysis.

### Time Interval Selection Buttons

Below the token list, you can choose a **time interval** for tracking price changes:

* **1h** – Changes over the last **hour**.
* **12h** – Changes over the last **12 hours**.
* **24h** – Changes over the last **24 hours**.

Tapping one of these buttons **updates the price and percentage change** for the selected time frame.

### “Hide Charts” Button

* **Toggles** the visibility of price change chart links.
