# ✏️ Edit

{% stepper %}
{% step %}
**Open the Main Menu** and tap on **“🔍 Tracking”**.
{% endstep %}

{% step %}
Select the category **“✏️ Edit”** and tap on **“⛽️ Gas Alerts”**.
{% endstep %}

{% step %}
You’ll see your current alert — you can:

* **Enter a new gas price value** to update the alert.
* Or choose one of the available actions below:
{% endstep %}
{% endstepper %}

<div align="left"><figure><img src="../../.gitbook/assets/Edit (2).png" alt="" width="375"><figcaption></figcaption></figure></div>

#### 📍 Available Actions:

* **🗑️ Delete alert** – Remove the gas price alert completely.
* **❌ Cancel** – Return to the previous menu without making changes.

{% hint style="success" %}
**Stay flexible** and **in control** of your **gas fee tracking** with **quick edits anytime**. 🚀
{% endhint %}
