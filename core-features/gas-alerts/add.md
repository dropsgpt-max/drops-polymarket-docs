# ➕ Add

{% stepper %}
{% step %}
**Open the Main Menu** and tap on **“🔍 Tracking”**.
{% endstep %}

{% step %}
Select the category **“➕ Add”** and tap on **“⛽️ Gas Alert”**.
{% endstep %}

{% step %}
Enter your **target gas price** (in Gwei)
{% endstep %}

{% step %}
Choose the **notification frequency**:

* **Onetime** – Alert triggers **once** when gas reaches the specified level. It is then automatically disabled.
* **Repeating** – Alert triggers **every time** gas hits the target, until you manually turn it off.
{% endstep %}
{% endstepper %}
