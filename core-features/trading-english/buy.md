# 🟢 Buy

#### 1. Buying Tokens

To buy tokens in the **Trading** section, you can:

* Open the **"Buy"** section and send the contract address or a link to the token.
* Send the contract or a link to the token in the bot chat, which will automatically open the purchase menu without the need to press additional buttons.

***

#### 2. Token Purchase Menu

1. 🔗 **Token Name (clickable)** – clicking opens the token page on **DexScreener**.
2. 📜 **Token Contract (clickable)** – copies the contract to the clipboard.
3. 📊 **Token Information Block:**
   * Token price in **USD**.
   * **Market Cap** (market capitalization).
   * **Liquidity** (available funds in the pool).
   * **24h Trading Volume**.
4. 🔄 **Swap Block:**
   * Displays the amount of **SOL** to be sent and the number of tokens received (with **USD** conversion in parentheses).
   * **Wallet Balance** – amount of **SOL** and its equivalent in **USD**.
5. ⚙️ **Interaction Buttons Block:**
   * ↩ **Back** – returns to the main trading menu.
   * 🔄 **Refresh** – updates token and swap information.
   * **Quick SOL Buttons** – presets for quick swap amount selection (configurable in trading settings).
   * ✏ **X SOL** – enter a custom swap value (**SOL** or percentage; last selected value is saved).
   * ⚙️ **Slippage** – slippage setting (can be changed in settings; last selected value is saved).
   * 👛 **Wallet** – option to select a different wallet or multiple wallets for the swap.
   * 🟢 **Buy** – confirms the token purchase.

***

#### 3. Post-Purchase Menu

🔹 **Main Block**

* **Wallet** used for the swap.
* **"Bought" (clickable)** – link to the transaction on the blockchain explorer.
* **Amount of purchased tokens** and their total value in **SOL** (converted to **USD**).
* **Updated wallet balance** after the purchase.

📊 **Current Position Information Block**

* **Token Name (clickable)** – displays the amount spent on token purchase (converted to **USD**) and opens the token card on **DexScreener**.
* **Quantity** – total number of tokens in the wallet.
* **PNL (Profit and Loss)** – displays profit or loss for the position:
  * 🟩 **Green Square** – if in profit.
  * 🟥 **Red Square** – if at a loss.
* Shows **percentage change**, **X multiplier** of profit/loss, and the **difference in SOL** (converted to **USD**).
* **Spent** – cost of purchase (**SOL** and **USD**).
* **Avg Entry** – average entry price (the price at which the token was bought).

📈 **Current Token Data Block**

* **Token Price**.
* **Market Cap** (market capitalization).
* **Liquidity** (available funds in the pool).
* **24h Trading Volume**.

🔘 **Interaction Buttons Block**

1. ↩ **Back** – returns to the main trading menu.
2. 🔄 **Refresh** – updates position information, current token price, and market parameters.
3. 📦 **Your Positions** – navigates to the section of all active positions, where you can view and manage purchased tokens.
4. 🔴 **Sell 100% ⚡** – instantly sells all tokens.
5. 🔴 **Sell** – opens the token selling menu.
6. 🟢 **Buy More** – opens the purchase menu for the same token.
7. 💰 **Add to Watchlist** – adds the token to the watchlist with default settings.
