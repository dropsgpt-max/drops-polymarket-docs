# 📤 Send

#### 1. **"Send" Section**

This section allows you to send tokens from the currently active wallet.

🔹 **Main Block**

* **Wallet Name (clickable)** – displays the currently active wallet.
* ✏ **/change\_tw** – option to switch the active wallet.
* **Wallet Address** – public address used for sending tokens.
* **Balance:** – total **SOL** balance (**in SOL and USD**).
* **Positions:** – total value of all tokens in the wallet (**in SOL and USD**).

📊 **Token List Block (Available for Sending)**

* A **list of all tokens** currently held in the wallet.
* For each token, the following details are displayed:
  * **Token Name (clickable)** – opens **DexScreener** for token details.
  * **Amount in SOL** – current token balance.
  * **Equivalent in USD** – token value in **USD**.

🔘 **Interaction Buttons Block**

* ↩ **Back** – returns to the previous menu.
* 🔄 **Refresh** – updates the list of available tokens and wallet balance.
* **Token Buttons** – select a token for sending.

***

#### 2. **Token Sending Menu**

🔹 **Main Block**

* **Token Name (clickable)** – links to **DexScreener** for token details.
* **Wallet Address** – public address used for sending.

📊 **Position Information Block**

* **Token Name (clickable)** – opens detailed token information.
* **Amount in SOL and USD** – current token balance in the wallet.
* **Quantity** – exact number of tokens available.

📌 **Current Balance Block**

* **Balance** – total **SOL** balance and its equivalent in **USD**.

🔘 **Interaction Buttons Block**

* ↩ **Back** – returns to the previous menu.
* 🔄 **Refresh** – updates balance and token data.
* **50%** – send **50%** of available tokens.
* **100%** – send **all tokens**.
* ✏ **X %** – enter a **custom amount** of tokens to send.
* **Wallet** – select the wallet from which tokens will be sent.
* **Receiver’s Address** – enter the recipient’s address.

***

#### 3. **Post-Token Sending Menu**

🔹 **Main Block**

* 📤 **Wallet Name** – displays the currently active wallet used for sending.
* **Sent (clickable)** – link to the transaction on the **blockchain explorer**.
* **Amount of Sent Tokens** – number of tokens sent.
* **Recipient’s Address (clickable)** – link to the **public address** of the recipient’s wallet.
* 💰 **Balance:** – updated wallet balance after sending (**in SOL and USD**).
* ↩ **Back** – returns to the **main menu**.
