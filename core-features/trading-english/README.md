# 💸 Trading (English)

#### 1. Getting Started with Trading

When you first enter the Trading section, a notification appears informing you that you have no wallets. You are offered two options:

* ➕ **Generate Wallet** – creates a new SOL wallet in the bot. The wallet is generated automatically and becomes available for management.
* 📥 **Import Wallet** – imports an existing SOL wallet using a private key.

**1.1 Importing Wallets**

After a successful wallet import, a notification appears, and three buttons become available:

* ✏ **Edit Wallet** – edit the imported wallet (change name, delete, etc.).
* ➕ **Add more Wallets** – add new wallets to the system.
* 👛 **My Wallets** – view the list of all imported wallets.

For bulk imports, instead of **Edit Wallet**, the **Edit Wallets** button appears, allowing you to edit ALL wallets at once.

**1.2 Generating Wallets**

You need to specify the number of wallets to be generated. After successful generation, a notification appears along with three buttons:

* ✏ **Edit Wallet** – edit the created wallet.
* ➕ **Add more Wallets** – add new wallets.
* 👛 **My Wallets** – view all available wallets.

***

#### 2. Main Trading Menu

**2.1 Selecting and Managing the Active Wallet**

* The currently active wallet is displayed with an option to open its page in **Solscan**.
* A button is available: ✏ **/change\_tw** to switch the active wallet.
* The full wallet address is displayed (clickable).

**2.2 Viewing Balance and Positions**

* Displays the total wallet balance in **SOL** and its equivalent in **USD**.
* Shows active positions (amount in **SOL** and **USD**).

**2.3 Interactive Buttons for Cryptocurrency Operations**

* 🟢 **Buy** – purchase tokens.
* 🔴 **Sell** – sell tokens.
* 📦 **Your Positions** – view current positions.
* 📤 **Send** – send tokens.
* 👛 **Your Wallets** – manage wallets.
* ⚙️ **Settings** – access settings.
