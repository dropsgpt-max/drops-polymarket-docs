# 💠 NFT Patterns

**NFT Pattern** is a powerful feature designed to track emerging trends across NFT collections by detecting **synchronized behavior among wallets**.

<div align="left"><figure><img src="../../.gitbook/assets/Alert (1).png" alt="" width="375"><figcaption></figcaption></figure></div>

### 🔍 What It Does

It monitors actions like **buying**, **selling**, or **minting** NFTs and notifies you when **multiple unique addresses** (e.g., whales, influencers, smart wallets) perform **the same action** on the **same collection** within a short time frame.

The idea is simple:

> The more unique wallets repeat the same action (buy, sell, mint) on a collection — the higher the chance it's a **potential gem**, a **pump signal**, or a **trend in the making**.
