### Available Settings in /EDIT Mode

<div align="left"><figure><img src="../../.gitbook/assets/EditOptions (1).png" alt="" width="375"><figcaption></figcaption></figure></div>

* **List** – Switch to another default list or edit your custom address list.
* **💸 Tx Value** – Set a **minimum transaction value (in ETH)**. Alerts won’t be triggered below this amount.
* **👨 NFT Age** – Filter collections by age to track only newly created or more established NFTs.
  * Available ranges:
    * **< 1 day**
    * **< 1 week**
    * **< 1 month**
    * **< 3 months**
    * **< 6 months**
    * **< 1 year**
    * **< 2 years**
    * **Any** – No age restrictions
* **✏️ Rename** – Rename the pattern for easier identification.
* **💠 Matches** – Set how many address matches are required to trigger an alert.
* **📶 Volume** – Define a **minimum collection trading volume**. Alerts will only trigger above this threshold.
* **📆 Event** – Choose which event types should trigger notifications:
  * **Buy**
  * **Sell**
  * **Mint**
  * **Free Mint**
* **🔈 Notifications** – Turn alerts **ON or OFF** for this specific pattern.
* **🗑 Delete** – Remove the NFT Pattern from your tracking list.
