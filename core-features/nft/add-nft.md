# ➕ Add NFT

{% stepper %}
{% step %}
**Open the Main Menu** and tap on **“🔍 Tracking”**.
{% endstep %}

{% step %}
Select the category **“➕ Add”** and tap on **“💎 NFT”**.
{% endstep %}

{% step %}
Enter the **NFT collection name** or **contract address** _(Ethereum network only)_.

Choose the desired collection from the list.
{% endstep %}

{% step %}
Set the **price change percentage** (e.g., 5%) or tap **“Skip”** if you don’t want price change alerts.

If you skip, you will **not receive alerts** for floor price changes.
{% endstep %}

{% step %}
Specify the **number of NFT buys/sells per minute** to trigger alerts.

Example: If you enter “1”, you’ll get a notification when there's **1 purchase or sale per minute**.
{% endstep %}
{% endstepper %}
